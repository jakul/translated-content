### Definición

**param** de "parameter" = parámetro. Permite especificar parámetros de ejecución para un objeto.

<dl><dd><strong>Sus etiquetas son</strong>: <code>&#x3C;param></code> (solo tiene una).</dd></dl>

<dl><dd><strong>Crea una caja</strong>: No.</dd></dl>

<dl><dd><strong>Está definido como</strong>: .</dd></dl>

<dl><dd><strong>Puede contener</strong>: Nada.<dl><dd><strong>Puede ser contenido por</strong>: El elemento <a href="es/HTML/Elemento/object">object</a>, y el desaprobado <a href="es/HTML/Elemento/applet">applet</a>.</dd></dl></dd></dl>
